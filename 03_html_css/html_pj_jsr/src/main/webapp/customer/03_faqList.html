3 결제/배송 Q [일반배송] 주문하면 언제 배송되나요?
A 이브온티는 평일 매일 오후 3시까지 주문완료시, 당일 출고 및 배송 출발합니다. 일반상품의 배송 택배사는 롯데택배입니다.

다만, 최근 택배기사님이 과다 업무처리하지 않도록 1일 출고량 제한이 있어서, 조금 이른 시간에 당일 배송출발이 마감될 수 있습니다. 이런 경우 주문 다음날 오후에 배송 출발합니다!

명절 등 긴 연휴 전에는 공지사항에서 배송일정을 확인해주세요!

1 회원가입/정보 Q [회원] 웰컴쿠폰 어떻게 사용하나요?
A 처음 가입하셨다면, 웰컴쿠폰으로 더 가볍게 차 한잔을 시작해보세요~!

[사용조건]
- 정기구독 제외한 모든 상품 주문시 사용 가능 -발금후 7일 이내 -15,000원 이상 구매시 사용

[사용방법]
-마음에 드는 상품을 구매하고 "주문서"에서 결제정보의 쿠폰내역 버튼을 눌러보세요 -> 팝업의 스크롤을 조금 내리면 쿠폰 발견 -> 선택 -개별 상품 할인쿠폰이 아니라 주문시 적용 쿠폰이라서 꼬옥 주문서에서 적용을 해주세요!
